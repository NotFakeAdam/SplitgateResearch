# Profile Block List

See who you have blocked why don't you!?

## Method `GET`

TOKEN: Auth Token.  

## Endpoint

`https://splitgate.accelbyte.io/lobby/v1/public/player/namespaces/splitgate/users/me/blocked`

## Headers

- `Authorization`: Bearer [TOKEN](../Auth/Auth%20Token.md)

## Response 

```json
{
 "data": []
}
```
